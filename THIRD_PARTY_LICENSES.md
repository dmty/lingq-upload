This bundle statically links the following components alongside source code
released under `MIT OR Apache-2.0`.

## libmp3lame (LGPL-2.1)

Used via the `mp3lame-encoder` crate. Upstream source:
https://lame.sourceforge.io/

The LGPL-2.1 grants you the right to relink the bundle against a modified
version of libmp3lame. The full project source — including build inputs and
the `mp3lame-encoder` crate — is published at
https://github.com/dmty/lingq-upload so that re-linking is possible without
contacting the authors. The LGPL-2.1 text is reproduced at
https://www.gnu.org/licenses/old-licenses/lgpl-2.1.txt.
